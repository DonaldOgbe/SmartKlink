# SmartKilnik

## Overview
SmartKilnik is a digital healthcare platform that enables users to consult a doctor, receive prescriptions, and access nearby pharmacies.

## Features
- AI-assisted doctor chat
- Prescription generation
- Pharmacy selection
- Payment integration

## Tech Stack
- Next.js
- Groq API (AI)
- Paystack

## How It Works
1. User enters details
2. Chats with AI doctor
3. Gets prescription
4. Selects pharmacy
5. Makes payment
